A real-time document editing app enabling multiple users to collaborate seamlessly with permission-based sharing.

Built with React and React Router for a dynamic frontend, Vite for optimized builds, and Go for a robust backend with WebSocket support. Deployed on Render with a clean, dark-themed UI, responsive textarea, and intuitive controls.

This project deepened my skills in React state management, Go API development, WebSocket communication, and production deployment.
